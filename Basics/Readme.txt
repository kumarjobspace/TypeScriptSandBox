This is a basic exercise to lean TypeScript
--------------------------------------------
1. Types
2. Functions/Methods
3. Interfaces
4. Classes


Setup ->
1. Make sure Node.js is installed
2. Install typescript using npm
    Command>npm install -g typescript
    Desc: -g stands for install globally
3. Run the below command to make tsc compiler run in watch mode
    command>tsc -w filename.tsc
    desc: w stand for watch mode
